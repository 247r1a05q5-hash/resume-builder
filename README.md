# AI Resume Evaluator

Scaffolded Next.js app with Tailwind CSS, Lucide icons, and mock AI/HR features.

Quick start:

1. Install deps:

```bash
npm install
```

2. Run dev:

```bash
npm run dev
```

This project includes:
- Landing page with role selection
- Seeker dashboard with chat, upload, and growth panels
- HR dashboard for bulk screening with mock processing
